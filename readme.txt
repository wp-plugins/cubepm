CubePM is a private messaging system that integrates right into the frontend of your site.

== Description ==

CubePM is a private messaging system for WordPress.

Unlike other PM system, CubePM allows messages to be sent to multiple users and also displays the whole PM system on the frontend using a short-code.

== Installation ==

Installing should be a piece of cake and take fewer than five minutes.

1. Upload, install and activate the plugin.
2. Create a new page and name it as you wish.
3. Add the following shortcode to the contents of the page: [cubepm]
4. Save the page and you are done setting up CubePM!
5. You may also wish to tweak how CubePM works by visiting the settings page.
